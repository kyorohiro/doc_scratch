# パズドラっぽいの

パズドラのパズルゲームの部分を作っていきましょう。


パズル&ドラゴンズ は、ガンホー・オンライン・エンターテイメント株式会社から配信されているパズルRPGゲームです。


![](about.png)

6x5　ないし 6x6 のマスに、ドロップが配置されていて、このドロップを縦横に3つ揃えると、
ドロップが消滅します。このドロップを効率よく消すパズルケームです。

ドロップが消滅した分だけモンスターへダメージを与える事ができ、
ゲームを進める事ができます。

# 上下(じょうげ)に移動(いどう)する

次(つぎ)は、上下(じょうげ)に移動(いどう)させてみましょう。これで、画面上(がめんじょう)を自由(じゆう)に動(うご)まわることが出来(でき)るようになります。


![](up_down_001a.png)

こんな感じです。左右への移動はX座標でした、上下の移動はY座標を変更します。

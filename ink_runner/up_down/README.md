# 上下(じょうげ)に移動(いどう)する

次(つぎ)は、上下(じょうげ)に移動(いどう)させてみましょう。これで、画面上(がめんじょう)を自由(じゆう)に動(うご)まわることが出来(でき)るようになります。


![](up_down_001a.png)

こんな感(かんじ)じです。左右(さゆう)への移動(いどう)はX座標(ざひょう)でした、上下(じょうげ)の移動(いどう)はY座標(ざひょう)を変更(へんこう)します。
